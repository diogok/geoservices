# Geoservices

Geospatial analysis and transformation as a service.

Expose JTS Simple Feature Specification(OGC) in a API of GEOJSon over HTTP.

## Features

Over Feature and FeatureCollection:

- reprojection
- buffer
- buffer in meters
- area
- area in meters
- union
